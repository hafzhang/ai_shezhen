# 中医舌诊知识库

## 一、舌诊基础理论

### 1.1 舌诊原理

舌诊是通过观察舌象（舌质、舌苔、舌形）来了解人体健康状况的诊断方法。

**舌诊的原理**：
- 舌体通过经络与五脏六腑相连
- 舌面分布反映不同脏腑的情况
- 舌质反映脏腑气血虚实
- 舌苔反映病邪性质与深浅

### 1.2 舌面脏腑分布

```
        ┌─────────────────────┐
        │        舌尖          │  → 心、肺
        ├─────────────────────┤
        │      舌两边          │  → 肝、胆
        ├─────────────────────┤
        │      舌中部          │  → 脾、胃
        ├─────────────────────┤
        │      舌根部          │  → 肾
        └─────────────────────┘
```

## 二、舌质诊断

### 2.1 舌质颜色

| 舌质 | 特征描述 | 临床意义 | 常见症状 |
|-----|---------|---------|---------|
| **淡白舌** | 舌色比正常浅白 | 气血不足、阳虚 | 面色苍白、乏力、畏寒、气短 |
| **淡红舌** | 颜色红活明润 | 正常健康舌象 | 无明显不适 |
| **红舌** | 舌色深于正常 | 实热或虚热 | 发热、口渴、便秘、尿黄 |
| **绛舌** | 深红色 | 热入营血 | 高热、夜寐不安、出血倾向 |
| **紫舌** | 舌色青紫 | 血瘀、寒凝 | 疼痛、肿块、口唇青紫 |
| **青舌** | 舌色如青筋 | 寒凝、血瘀 | 畏寒肢冷、剧烈疼痛 |

### 2.2 舌形形态

| 舌形 | 特征 | 临床意义 | 调理建议 |
|-----|------|---------|---------|
| **正常** | 柔软灵活 | 气血调和 | 保持健康生活方式 |
| **胖大** | 舌体胖大 | 阳虚水停、痰湿内盛 | 温阳化湿、清淡饮食 |
| **瘦薄** | 舌体瘦小 | 阴虚、气血不足 | 滋阴养血、营养丰富饮食 |
| **齿痕** | 舌边有齿印 | 脾虚湿盛 | 健脾祛湿、规律作息 |
| **裂纹** | 舌面有裂沟 | 阴血不足、热盛伤阴 | 滋阴润燥、多饮水 |
| **芒刺** | 舌面有红刺 | 热毒内盛 | 清热解毒、清淡饮食 |
| **强硬** | 舌体僵硬 | 热入心包或中风 | 及时就医诊治 |

## 三、舌苔诊断

### 3.1 苔色

| 苔色 | 临床意义 | 常见证型 | 调理方向 |
|-----|---------|---------|---------|
| **白苔** | 表证、寒证 | 风寒表证、里寒证 | 辛温解表、温里散寒 |
| **黄苔** | 里热证 | 实热、湿热 | 清热泻火、清热化湿 |
| **灰苔** | 里热或里寒 | 里热实证、里寒实证 | 根据寒热辨证 |
| **黑苔** | 重证 | 极热或极寒 | 急需专业诊治 |

### 3.2 苔质

| 苔质 | 特征 | 临床意义 |
|-----|------|---------|
| **薄苔** | 苔薄能见舌质 | 正常或表证、轻证 |
| **厚苔** | 苔厚不见舌质 | 里证、病邪较重 |
| **润苔** | 苔面湿润 | 津液未伤 |
| **燥苔** | 苔面干燥 | 津液已伤 |
| **腻苔** | 苔质颗粒细腻致密 | 湿浊、痰饮、食积 |
| **腐苔** | 苔质颗粒粗大疏松 | 食积、痰浊 |
| **剥苔** | 舌面部分剥落 | 胃气不足、胃阴亏损 |
| **无苔** | 舌面光洁无苔 | 胃气枯竭、胃阴大伤 |

### 3.3 常见舌苔组合

| 组合 | 临床意义 | 建议调理 |
|-----|---------|---------|
| 薄白苔 | 正常或风寒表证 | 注意保暖，多饮水 |
| 白厚腻苔 | 寒湿、痰湿、食积 | 温阳化湿、消食导滞 |
| 薄黄苔 | 里热初起 | 清热泻火 |
| 黄厚腻苔 | 湿热、痰热 | 清热化湿 |
| 少苔/无苔 | 阴虚、胃气不足 | 滋阴养胃 |
| 花剥苔 | 胃气阴两伤 | 益气养阴 |

## 四、综合辨证示例

### 4.1 常见证型舌象

#### 气虚证
- **舌象**：淡白舌、胖大有齿痕、薄白苔
- **症状**：乏力、气短、自汗、易疲劳
- **调理**：补气健脾（如四君子汤）

#### 血虚证
- **舌象**：淡白舌、瘦薄、薄白苔
- **症状**：面色苍白、头晕眼花、心悸失眠
- **调理**：补血养血（如四物汤）

#### 阴虚证
- **舌象**：红舌、瘦薄、少苔或无苔
- **症状**：口干咽燥、五心烦热、盗汗
- **调理**：滋阴降火（如六味地黄丸）

#### 阳虚证
- **舌象**：淡白舌、胖大有齿痕、湿润
- **症状**：畏寒肢冷、面色苍白、小便清长
- **调理**：温阳散寒（如金匮肾气丸）

#### 湿热证
- **舌象**：红舌、黄腻苔
- **症状**：口苦口黏、小便黄赤、大便黏滞
- **调理**：清热利湿（如龙胆泻肝汤）

#### 痰湿证
- **舌象**：淡白或淡红舌、白腻苔
- **症状**：胸闷痰多、头身困重、食欲不振
- **调理**：燥湿化痰（如二陈汤）

#### 瘀血证
- **舌象**：紫舌或有瘀点瘀斑
- **症状**：疼痛固定、肿块、出血色紫暗
- **调理**：活血化瘀（如血府逐瘀汤）

### 4.2 舌诊与季节关系

| 季节 | 正常舌象变化 | 注意事项 |
|-----|-------------|---------|
| 春季 | 舌略偏红 | 注意肝气舒畅 |
| 夏季 | 舌偏红、苔略黄 | 清热解暑 |
| 长夏 | 舌偏淡、苔腻 | 化湿醒脾 |
| 秋季 | 舌偏干、少苔 | 滋阴润燥 |
| 冬季 | 舌淡、苔薄白 | 温阳保暖 |

### 4.3 舌诊与体质关系

| 体质类型 | 典型舌象 | 调理原则 |
|---------|---------|---------|
| 平和质 | 淡红舌、薄白苔 | 保持平衡 |
| 气虚质 | 淡白舌、齿痕 | 补气健脾 |
| 阳虚质 | 淡白舌、胖大湿润 | 温阳散寒 |
| 阴虚质 | 红舌、少苔 | 滋阴降火 |
| 痰湿质 | 淡红舌、白腻苔 | 化痰祛湿 |
| 湿热质 | 红舌、黄腻苔 | 清热利湿 |
| 血瘀质 | 紫舌、瘀点 | 活血化瘀 |
| 气郁质 | 淡红舌、舌边略红 | 疏肝解郁 |

## 五、舌诊操作规范

### 5.1 观察条件

| 项目 | 要求 |
|-----|------|
| 光线 | 自然充足光线，避免有色光干扰 |
| 时间 | 最好在上午，避免进食后立即观察 |
| 姿势 | 正坐，自然伸舌，舌体放松 |
| 顺序 | 先舌尖后舌根，先舌质后舌苔 |

### 5.2 注意事项

1. **伸舌时间**：不宜过长，一般不超过10秒
2. **避免染色**：观察前避免食用有色食物
3. **自然状态**：舌体自然放松，不要用力伸舌
4. **综合判断**：舌象需结合其他症状综合分析

### 5.3 常见干扰因素

| 因素 | 影响 | 处理方法 |
|-----|------|---------|
| 饮食染色 | 染色苔 | 询问饮食史，清除后观察 |
| 药物影响 | 苔色改变 | 了解用药史 |
| 饮水过少 | 苔燥 | 询问饮水情况 |
| 吸烟 | 黄褐苔 | 了解吸烟史 |
| 情绪激动 | 舌红 | 安静后再观察 |
| 光线不足 | 舌色判断不准 | 充足光线下观察 |
