## 一、算法概述

### 1.1 算法基本信息

| 属性 | 描述 |
|------|------|
| **算法名称** | 基于深度学习的舌象分割与分类算法 |
| **算法版本** | v2.3 |
| **算法类型** | 语义分割 + 多标签分类 |
| **预期用途** | 辅助中医师进行舌象分析和辨证参考 |
| **适用人群** | 具有中医执业资格的医师 |
| **禁忌症** | 不能作为独立诊断依据，紧急情况需立即就医 |

### 1.2 算法架构

本系统采用双阶段深度学习架构：

```
输入图像 (舌部照片)
    ↓
第一阶段：舌体分割 (BiSeNetV2-STDCNet2)
    ↓
分割Mask (舌体轮廓)
    ↓
第二阶段：多标签分类 (PP-HGNetV2-B4)
    ↓
分类结果 (6维度18类)
    ↓
规则库/LLM诊断
    ↓
诊断报告 (证型分析)
```

### 1.3 核心算法组件

| 组件 | 模型架构 | 输入 | 输出 | 参数量 |
|------|----------|------|------|--------|
| **分割模型** | BiSeNetV2-STDCNet2 | 舌部图像 | 舌体mask | 15.2M |
| **分类模型** | PP-HGNetV2-B4 | 舌体图像 | 18维特征向量 | 18.7M (FP16: 8.7M) |
| **规则引擎** | 特征加权匹配 | 18维特征 | 证型概率 | N/A |
| **LLM诊断** | 文心一言ERNIE-Speed | 结构化特征 | 诊断报告 | N/A |

## 四、算法可解释性

### 4.1 分割结果可视化

**实现方式**: 舌体分割mask叠加在原图上

```python
from models.pipeline.segmentation import TongueSegmentationPredictor

predictor = TongueSegmentationPredictor(model_path='best_model.pdparams')
result = predictor.predict(image='tongue.jpg', return_overlay=True)
# result['overlay'] 显示分割区域（红色半透明）
```

**可视化效果**:
- 原图：显示原始舌部图像
- Mask叠加：舌体区域用红色半透明标注
- 轮廓线：绿色轮廓线标示边界
- 对比视图：三栏对比（原图|mask|叠加）

### 4.2 分类热力图 (Grad-CAM)

**实现方式**: 梯度加权类激活映射

```python
from models.paddle_clas.evaluation import ClassificationEvaluator

evaluator = ClassificationEvaluator(model, head_configs)
heatmap = evaluator.generate_gradcam_visualization(
    image=image_np,
    target_class=2,
    head_name='tongue_color',
    save_path='gradcam.png'
)
```

**热力图含义**:
- 红色区域：高度激活（模型关注区域）
- 黄色区域：中度激活
- 蓝色区域：低度激活

### 4.3 特征贡献度分析

**实现方式**: 计算各分类头对最终结果的重要性

```python
from models.paddle_clas.evaluation import FeatureContribution

importance = FeatureContribution.compute_head_importance(model, input_tensor)
explanation = FeatureContribution.generate_explanation(
    head_name='tongue_color',
    predicted_class=1,
    class_names=['淡白舌', '淡红舌', '红舌', '绛紫舌'],
    confidence=0.85,
    feature_importance=importance

```

**输出示例**:
```

特征贡献分析：
- 舌色维度：高贡献度（权重0.25）
- 苔色维度：中贡献度（权重0.20）
- 舌形维度：低贡献度（权重0.10）
```

## 五、算法局限性

### 5.1 已知局限

| 局限类型 | 描述 | 风险等级 | 缓解措施 |
|----------|------|----------|----------|
| **光照敏感性** | 强光/弱光影响分割精度 | 中 | 图像增强、多场景训练 |
| **少数类识别** | 绛紫舌、黑苔等少数类召回率较低 | 中 | Focal Loss、过采样 |
| **舌体变异** | 特殊舌形（裂纹舌、地图舌）识别不稳定 | 中 | 增加罕见样本 |
| **中医差异** | 不同中医流派的辨证差异 | 低 | 专家共识标注 |
| **设备依赖** | 不同拍摄设备的色差影响 | 低 | 色彩校正算法 |

### 5.2 适用场景限制

**适用场景**:
- ✅ 中医医疗机构辅助诊断
- ✅ 健康管理中心的舌象分析
- ✅ 中医教学和科研

**不适用场景**:
- ❌ 作为独立诊断依据
- ❌ 紧急医疗情况
- ❌ 无中医执业资格的人员使用

### 5.3 性能边界

| 指标 | 实际值 | 边界条件 | 说明 |
|------|--------|----------|------|
| **最小分辨率** | 512×512 | 低于256×256性能下降 | 分割精度显著降低 |
| **最大分辨率** | 4096×4096 | 高于4096推理超时 | 内存限制 |
| **并发QPS** | 100 | 高于150响应延迟 | CPU资源限制 |
| **准确率** | 87.3% | 少数类<70% | 类别不平衡影响 |
