## Codebase Patterns
- The dataset uses COCO format with train/val/test splits
- Image files have irregular filenames (with parentheses like 'F (1).jpg')
- Categories are defined in classes.txt files with Chinese names
- Category IDs start from 0 (jiankangshe) to 20 (xinfeitu)
- The annotation file may reference category_id 21 which is not defined in categories
