**Learnings for future iterations:**
- Patterns discovered: FastAPI使用lifespan上下文管理器进行启动/关闭处理; Pydantic v2使用pydantic-settings而不是pydantic.env_settings
- Patterns discovered: Base64图像编码/解码需要处理data URL前缀（如"data:image/png;base64,"）
- Gotchas encountered: Windows console需要UTF-8编码包装器处理中文字符; pydantic-settings需要extra="ignore"忽略未知环境变量
- Gotchas encountered: PIL Image与OpenCV numpy数组的颜色空间转换需要注意(RGB<->BGR)
- Useful context: API响应格式需要包含success字段、message/error字段、以及业务结果字段
- Useful context: 审计日志应该使用单独的logger和文件handler，便于日志解析和留存管理
- Useful context: FastAPI的exception_handler可以统一处理RequestValidationError和Exception
- Useful context: Swagger文档通过FastAPI的docs_url和redoc_url参数自动生成

**Usage:**
```bash
# 安装依赖
pip install -r api_service/requirements.txt

# 配置环境变量
cp api_service/.env.example api_service/.env
# 编辑 .env 文件设置模型路径和API密钥

# 启动服务
python -m api_service.app.main
# 或使用启动脚本
python api_service/start_api.py --port 8000
```

**API文档:**
访问 http://localhost:8000/docs 查看完整的Swagger UI文档
