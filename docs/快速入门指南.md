
#### 方案A：EasyDL + 文心一言

1. 访问 https://cloud.baidu.com/ 注册百度账号
2. 开通以下服务：

#### 方案B/C：PaddlePaddle

1. 安装PaddlePaddle框架
2. 下载预训练模型

### Step 3: 准备数据

**数据要求**：
- 图片格式：JPG、PNG
- 图片大小：建议512x512像素以上
- 拍摄环境：自然光，背景简洁
- 数据量：每类至少100张

**数据分类**：
```
舌象分类/
├── 淡白舌/
│   ├── img_001.jpg
│   ├── img_002.jpg
│   └── ...
├── 红舌/

└── ...
```

### Step 4: 训练/配置模型

#### 使用EasyDL训练：
1. 登录EasyDL控制台
2. 创建图像分类任务
3. 上传标注数据
4. 点击训练
5. 部署为API

#### 使用PaddlePaddle训练：


### Step 5: 调用API


## 二、三种方案详细对比

### 方案A：云端API方案（最简单）

**适合人群**：不想写代码、快速验证想法

**工作流程**：
```
上传图片 → EasyDL分类 → 文心一言分析 → 返回结果
```

**优势**：
- ✅ 零代码操作
- ✅ 部署快速
- ✅ 自动维护

**劣势**：
- ❌ 数据需上传云端
- ❌ 有调用费用
- ❌ 需要网络

**预估成本**：
- EasyDL：按调用量计费，约0.01元/次
- 文心一言：按Token计费，约0.008元/千Token
- 月均（1000次调用）：约20-50元

### 方案B：混合方案（推荐）

**适合人群**：有开发能力，需要平衡性能和成本

**工作流程**：
```
本地图片 → 本地分割/分类 → 文心一言分析 → 返回结果
```

**优势**：
- ✅ 图像处理本地化
- ✅ 智能分析由AI完成
- ✅ 成本相对较低
- ✅ 部分数据不上传

**预估成本**：
- 服务器：300-1000元/月
- API调用：5-20元/月
- 月均（1000次调用）：约300-1000元

### 方案C：纯本地方案

**适合人群**：医疗机构，注重隐私

**工作流程**：
```
本地图片 → 本地全流程处理 → 本地知识库 → 返回结果
```

**优势**：
- ✅ 完全本地化
- ✅ 数据隐私保护
- ✅ 无网络依赖
- ✅ 长期成本低

**劣势**：
- ❌ 开发工作量大
- ❌ 需要硬件投入
- ❌ 智能程度依赖规则库

**预估成本**：
- 服务器：1000-3000元/月（含GPU）
- 开发成本：1-2个月人力

## 三、百度AI服务快速对接

### 3.1 文心一言API对接

**1. 获取密钥**
```
访问：https://qianfan.cloud.baidu.com/
注册 → 创建应用 → 获取API Key和Secret Key
```

**2. 调用示例**
```python
import requests

# 获取Access Token
auth_url = "https://aip.baidubce.com/oauth/2.0/token"
params = {
    "grant_type": "client_credentials",
    "client_id": "你的API Key",
    "client_secret": "你的Secret Key"
}
response = requests.post(auth_url, params=params)
access_token = response.json()["access_token"]

# 调用文心一言
chat_url = "https://aip.baidubce.com/rpc/2.0/ai_custom/v1/wenxinworkshop/chat/ernie-speed"
headers = {"Content-Type": "application/json"}
data = {
    "messages": [
        {"role": "user", "content": "舌质红、苔薄黄，请分析健康状况"}
    ]
}
params = {"access_token": access_token}

response = requests.post(chat_url, headers=headers, json=data, params=params)
print(response.json())
```

### 3.2 EasyDL图像分类对接

**1. 创建模型**
```
登录 https://cloud.baidu.com/product/easydl
创建图像分类任务 → 上传数据 → 训练 → 发布
```

**2. 调用API**
```python
import base64
import requests

# 图片转base64
with open("tongue.jpg", "rb") as f:
    image_base64 = base64.b64encode(f.read()).decode()

# 调用预测API
url = "https://aip.baidubce.com/rpc/2.0/ai_custom/v1/classification"
params = {"access_token": "你的access_token"}
data = {"image": image_base64}

response = requests.post(url, params=params, json=data)
print(response.json())
```

```python
model = PaddleSegModel(model_type='seg')

result = model.predict('tongue_image.jpg')
```

```python
result = clas.predict('tongue_image.jpg')

```

## 五、数据采集指南

### 5.1 拍摄规范

**环境要求**：
- 光线：自然光或专业摄影灯
- 背景：纯色（白/灰/蓝）
- 设备：手机或相机，500万像素以上

**拍摄方法**：
```
1. 受检者自然坐姿
2. 张口自然伸出舌头
3. 相机正对舌体
4. 保持适当距离（20-30cm）
5. 避免闪光灯直射
```

**注意事项**：
- 拍摄前避免食用染色食物
- 避免刚喝完热水或冷饮
- 舌体不要过于用力伸出
- 同一位置拍摄多张备用

### 5.2 数据标注工具

**推荐工具**：

| 工具 | 特点 | 下载 |
|-----|------|------|
| LabelImg | 轻量级，适合分类标注 | GitHub免费 |
| CVAT | 功能强大，适合复杂标注 | 在线免费 |
| 百度智能云标注 | 专业标注服务 | 付费 |

**标注规范**：
```
舌质颜色：淡白/淡红/红/紫/青
舌苔：薄白/白厚/薄黄/黄厚/少苔/无苔/灰黑
舌形：正常/胖大/瘦薄/齿痕/裂纹/点刺
```

## 六、常见问题排查

### Q1: API调用失败

**可能原因**：
- API Key错误
- Access Token过期
- 调用频率超限

**解决方法**：
```python
# 检查API Key是否正确
print(api_key)

# 重新获取Access Token
token = get_new_token()

# 检查账户余额
# 登录控制台查看
```

### Q2: 模型识别不准

**可能原因**：
- 训练数据不足
- 数据质量差
- 模型参数不当

**解决方法**：
- 增加训练数据（每类500+）
- 提高图片质量
- 调整模型参数
- 使用数据增强

### Q3: 速度太慢

**优化方法**：
- 使用轻量级模型
- 减小图片尺寸
- 启用GPU加速
- 使用模型量化
