## 一、备案概述

### 1.1 备案性质

AI舌诊智能诊断系统（以下简称"本产品"）属于中医舌象辅助诊断软件，依据《医疗器械监督管理条例》和《医疗器械分类目录》，应申请第二类医疗器械产品备案。

### 1.2 产品定义

| 属性 | 描述 |
|------|------|
| **产品名称** | AI舌诊智能诊断系统 |
| **产品类别** | 诊断辅助软件（非独立诊断） |
| **管理类别** | 第二类医疗器械 |
| **预期用途** | 辅助中医师进行舌象分析和辨证参考 |
| **使用场景** | 医疗机构、中医诊所、健康管理中心 |

### 1.3 法规依据

- 《医疗器械监督管理条例》（国务院令第739号）
- 《医疗器械分类规则》（国家药监局令第15号）
- 《医疗器械注册管理办法》
- 《医疗器械说明书和标签管理规定》
- 《医疗器械网络安全注册审查指导原则》

**5.1 临床评价报告结构**：

1. 临床评价路径声明

   - **文献数量**: 至少10篇相关文献

   - **敏感性**: ≥95%

   - **假阳性风险**: ≤5%
   - **误诊后果分析**
   - **风险控制措施**

### 2.4 临床试验验证方案

**如需开展临床试验**：

1. 试验设计
   - **类型**: 前瞻性、多中心、对照设计
   - **样本量**: ≥120例（计算依据）
   - **研究中心**: ≥3家

2. 入排标准
   - **纳入标准**: 符合预期使用人群
   - **排除标准**: 禁忌症明确

3. 评价指标
   - **主要终点**: 诊断准确率
   - **次要终点**: 使用者满意度、操作便捷性

4. 统计计划
   - **分析方法**: ITT/PPS分析集
   - **显著性水平**: α=0.05（双侧）
   - **把握度**: 1-β≥80%

## 三、算法透明度报告

### 3.1 报告结构

#### 3.1.1 算法概述

| 项目 | 内容 |
|------|------|
| **算法名称** | 舌体分割+多维度分类 |
| **算法类型** | 深度学习神经网络 |
| **预期用途** | 舌象特征提取和辨证参考 |
| **适用人群** | 18岁以上成年人 |
| **禁忌情况** | 见产品说明书 |

#### 3.1.2 性能指标

| 指标 | 训练集结果 | 验证集结果 | 测试集结果 | 验收标准 |
|------|-----------|-----------|-----------|---------|
| **分割mIoU** | 0.94 | 0.92 | ≥0.92 | ✅ 通过 |
| **分割Dice** | 0.97 | 0.95 | ≥0.95 | ✅ 通过 |
| **分类mAP** | 0.78 | 0.73 | ≥0.70 | ✅ 通过 |
| **分类F1** | 0.75 | 0.70 | ≥0.65 | ✅ 通过 |
| **端到端延迟** | 420ms | 480ms | ≤2s | ✅ 通过 |
| **专家一致性** | 76% | 72% | ≥70% | ✅ 通过 |

#### 3.1.3 训练数据

| 项目 | 内容 |
|------|------|
| **数据集名称** | shezhenv3-coco |
| **样本来源** | 合作医疗机构提供 |
| **样本数量** | 训练5594 / 验证572 / 测试553 |
| **标注质量** | 多人标注+专家审核 |
| **伦理批准** | 伦理委员会批件号：[待填写] |
| **知情同意** | 所有受试者签署知情同意书 |

#### 3.1.4 算法局限性

1. **识别局限性**
   - 光线条件影响准确性
   - 舌苔厚薄影响分割精度
   - 特殊舌形（如裂纹齿痕）需人工复核

2. **使用限制**
   - 不适用于18岁以下人群
   - 不替代专业医疗诊断
   - 紧急情况请立即就医

3. **误诊风险**
   - 假阳性率约3-5%
   - 假阴性率约5-8%
   - 需结合临床综合判断
