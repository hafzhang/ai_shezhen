1. **特征不完整**: 18.2%的示例缺少2个以上主要特征

3. **特征-证型不一致**: 部分示例特征与典型证型表现有偏差
   - 例如: 肝胆湿热证部分示例为白苔（典型应为黄苔）

4. **健康舌标注**: 所有示例标注为"非健康舌"，与证型诊断一致
   - ✓ 无健康舌被错误诊断为证型的案例
