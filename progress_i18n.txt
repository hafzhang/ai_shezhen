**Learnings for future iterations:**
- Patterns discovered: Use vue-i18n Composition API mode (legacy: false) for Vue 3
- Patterns discovered: Create type-safe locale files using 'as const' for better type inference
- Patterns discovered: Store locale preference in localStorage with key 'app_language'
- Patterns discovered: Use composables for language management (useLanguage pattern)
- Gotchas encountered: vue-tsc is incompatible with Node.js v24, use Node.js v20 for type checking
- Gotchas encountered: Browser language detection uses navigator.language, check prefix with startsWith()
- Gotchas encountered: Vite doesn't require .ts extensions in imports for module resolution
- Useful context: vue-i18n v11.x requires `legacy: false` for Composition API mode
- Useful context: Use `globalInjection: true` to enable $t() in templates
- Useful context: Language preference should be initialized on app mount for immediate effect

**Usage:**
